  1. 1. crie um repositorio no GitHub com o mesmo nome do seu projeto
  2. 2. No terminal, inicialize o repositorio local com git init.
  3. 3. Altere o nome da branch principal para main
  4. 4. Adicione os arquivos com git add. e faça o commit com git commit -m "first commit"
  5. 5. Conecte ao repositório remoto com: git remote add origin https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
  6. 6. Envie os arquivos com git push -u origin main
  7. 7. No GitHub, vá em Settings > Pages
  8. 8. Em "branch"
